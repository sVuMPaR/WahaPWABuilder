# Waha PWA Builder

Офлайн-конструктор армейских списков Warhammer 40k (11-е издание). Данные из [Wahapedia](https://wahapedia.ru/) и [MFM](https://mfm.warhammer-community.com/).

**Открыть приложение:** [https://svumpar.github.io/WahaPWABuilder/](https://svumpar.github.io/WahaPWABuilder/)

## Возможности

- Каталог фракций с очками MFM
- **Карточки юнитов**: превью характеристик (M, T, Sv, W, Ld, OC) под названием; по клику — полная информация (способности, профили оружия, loadout builder)
- **Парсинг wargear options**: Wahapedia-опции разбираются в интерактивный loadout builder (замены, опции, MFM-очки); нераспознанные — текстом
- Сборка ростера: детачменты, enhancements (до 3), привязка лидеров, экспорт/копирование списка
- **Свой лимит очков** (250–10000) для казуальных игр
- **Проверка правил армии**: Epic Hero максимум 1, Battleline максимум 6, остальные юниты максимум 3
- **Метки Bodyguard** по связям Wahapedia и полю `attachTo` из MFM
- PWA: офлайн после явной загрузки данных (см. **Prepare offline** ниже)

## Офлайн-режим

Приложение **не скачивает все армии автоматически**. Ростеры хранятся **только на вашем устройстве** (IndexedDB в браузере). На сервер ничего не отправляется.

### Установка на телефон (рекомендуется)

1. Откройте [https://svumpar.github.io/WahaPWABuilder/](https://svumpar.github.io/WahaPWABuilder/) в Chrome (Android) или Safari (iOS).
2. **Добавьте на экран** / «Установить приложение» — удобнее открывать, но **это не заменяет загрузку данных** (шаг ниже).

### Обязательно: один раз загрузить данные онлайн

Офлайн доступны **только те фракции, которые вы скачали на устройство**, пока был интернет. Каталог фракций тоже нужно получить хотя бы раз онлайн.

**Рекомендуемый способ — Prepare offline:**

1. Откройте приложение при включённом интернете.
2. На странице **Factions** или **Rosters** нажмите **Prepare offline**.
3. Выберите нужные армии (кнопка **From rosters** подставит фракции из ваших списков).
4. Нажмите **Download selected** и дождитесь завершения — у скачанных армий появится бейдж **Cached**.

**Альтернатива:** вручную открыть каждую нужную фракцию в разделе **Factions** и дождаться списка юнитов.

После загрузки можно отключить интернет: каталог, скачанные фракции и ваши ростеры останутся доступны.

### Что работает офлайн

| Действие | Офлайн |
|----------|--------|
| Каталог фракций (после первого онлайн-визита или Prepare offline) | ✅ |
| Фракция, скачанная через **Prepare offline** или открытая онлайн | ✅ |
| Редактирование сохранённых ростеров | ✅ |
| Фракция, которую **не скачивали** | ❌ — экран с ошибкой, нужен интернет |
| Обновление данных Wahapedia/MFM | ❌ — только онлайн |

### Если что-то не грузится

- **Экран с ошибкой про офлайн-данные** — фракция не скачана. Подключите интернет → **Prepare offline** → скачайте эту армию (или откройте её в **Factions**).
- **Prepare offline недоступен / ничего не качается** — проверьте интернет и дождитесь окончания загрузки (прогресс-бар).
- **Space Marines** — большой пак (~2.7 MB); на слабом соединении подождите 15–30 секунд.
- **Ростеры пропали** — они привязаны к браузеру и устройству; очистка данных сайта или другой браузер = пустой список.

### Краткая памятка

```
Онлайн:  Prepare offline → выбрать армии → Download selected → дождаться Cached
Офлайн:  скачанные фракции + сохранённые ростеры работают без сети
```
